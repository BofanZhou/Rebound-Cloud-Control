# 回弹云控管理系统 —— 项目介绍

## 一、项目概述

**回弹云控管理系统**（Rebound Cloud Control）是一款面向钢管折弯加工场景的智能化 Web 控制平台。系统致力于解决钢管折弯过程中因材料弹性回弹导致的加工精度偏差问题，通过数据驱动的参数推荐与 AI 模型训练，帮助操作员快速获取最优补偿参数，提升加工精度与生产效率。

> **愿景：让每一位折弯操作员都能像资深工艺师一样，精准掌控回弹补偿。**

---

## 二、核心问题

在钢管折弯加工中，材料在撤去外力后会发生**弹性回弹**，导致实际折弯角度偏离目标值。传统解决方案依赖老师傅的经验估算，存在以下痛点：

- **精度不稳定**：人工估算误差大，废品率高
- **经验难传承**：老师傅退休，工艺知识流失
- **设备管理混乱**：多台设备状态不透明，切换繁琐
- **数据无沉淀**：历史加工数据散落，无法用于持续优化

## 四、核心功能

### 1. 双模式登录
- **机器登录**：直接输入机器编号，一键进入单设备控制界面
- **用户登录**：支持管理员、维修员、操作员三种角色，登录后选择目标机器

### 2. 设备状态监控
- 实时查看设备运行状态（运行中 / 待机 / 故障 / 离线）
- 显示温度、压力、当前角度、补偿量等关键工艺参数
- WebSocket 实时推送，状态变化秒级同步

### 3. 回弹参数推荐
- 输入材质、管径、壁厚、目标角度等参数
- 系统基于规则引擎与历史数据，智能计算补偿值
- 一键下发推荐参数至设备执行

### 4. 历史记录管理
- 自动记录每次折弯作业的输入参数、推荐值、实际结果
- 支持按机器、时间、材质等多维度筛选与导出
- 为模型训练提供高质量标注数据

### 5. AI 模型训练
- **数据集上传**：支持 CSV 格式历史数据导入
- **在线训练**：基于 PyTorch 搭建神经网络，自动训练回弹分类模型
- **训练可视化**：实时展示 Loss 曲线与验证准确率
- **迭代预测**：输入目标参数，系统自动迭代逼近最优补偿角

### 6. 多机管理
- 添加、编辑、删除多台生产设备
- 机器状态统一管理，支持快速切换当前操控目标
- 每台机器拥有独立的模拟器与数据隔离

### 7. 用户与权限管理
- 管理员可创建、删除用户，重置密码
- 操作日志全程记录，支持审计追溯

## 五、应用场景

| 场景 | 价值 |
|------|------|
| **小型加工厂** | 低成本部署，替代老师傅经验，快速上手 |
| **多机生产车间** | 统一管控多台折弯机，减少切换成本 |
| **工艺研发部门** | 沉淀数据，训练专属模型，持续优化工艺 |
| **职业院校实训** | 直观展示回弹原理与补偿逻辑，辅助教学 |

## 六、项目愿景

### 短期目标
- 打造可本地一键运行的完整原型系统
- 验证规则引擎 + AI 模型在回弹补偿场景的可行性
- 积累真实加工数据，持续迭代模型精度

### 中期目标
- 接入真实 PLC / 数控系统，实现参数自动下发
- 支持更多材质与复杂截面钢管
- 构建云端模型训练平台，支持跨工厂数据共享

### 长期愿景
> **成为中国钢管折弯行业智能化升级的标配工具，让回弹补偿从"老师傅手艺"变为"数据科学"。**
