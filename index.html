01 / ABOUT

About Project 关 于 项 目

Rebound Cloud Control is an intelligent compensation system for steel pipe springback in bending operations. It bridges the gap between theoretical parameters and real-world manufacturing deviations.

The system takes pipe specifications — diameter, thickness, material, and target angle — and computes optimal compensation parameters through a rule-based engine. Each machine operates independently with isolated data storage and full history tracking.

With multi-machine management and role-based authentication, the platform is designed to scale from a single workshop prototype to a distributed production environment.

02 / TECH

Technology 技 术 栈

Vue 3

Progressive frontend framework with Composition API for reactive state management and component architecture.

Vite

Next-generation build tooling with instant HMR, optimized for modern web development workflows.

TypeScript

Type-safe development across the entire stack, from API contracts to UI components.

FastAPI

High-performance Python backend with automatic OpenAPI documentation and Pydantic validation.

ECharts

Powerful charting library for real-time device monitoring and historical data visualization.

Pinia

Intuitive store pattern for Vue with devtools support and TypeScript-friendly API design.

Simulator

Virtual PLC device simulator for local demo and testing without hardware dependencies.

Auth

JWT-based authentication with machine login and role-based user access control.

03 / FEATURES

Features 核 心 功 能

03.A

Parameter Input

Input pipe diameter, wall thickness, material type, and target bending angle through an intuitive form interface with real-time validation.

03.B

Smart Recommendation

Rule-based engine computes recommended compensation angle and offset values based on material properties and geometric parameters.

03.C

Device Monitoring

Real-time status display with online/offline indicators, current angle readings, and deviation analysis with automatic refresh.

03.D

Multi-Machine

Manage unlimited machine nodes with isolated data storage. Switch between devices seamlessly from a unified dashboard.

03.E

History Tracking

Complete audit trail of all submitted tasks with input parameters, recommendations, execution results, and timestamps.

03.F

Role System

Three-tier access control — Administrator, Maintenance, and Operator — with differentiated permissions and capabilities.
